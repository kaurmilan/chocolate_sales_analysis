#Chocolate Sales Data Analysis

#Overview:
This project analyses chocolate sales across multiple countries over three years.

#Objective:
The goal was to identify sales trends, seasonal patterns and product perfomance using python and Power BI.

#Dataset
-Source: Kaggle.com (Chocolate sales - Saidamin Saidakhmadov)
-Key columns: Sales Person,Country,Product,Date,Amount,Boxes Shipped

#Tools and technologies:
-Python(pandas,maplotlib)
-Jupyter Notebook(VS Code)
-Power BI

#Data cleaning and preparation
-Removed null values
-Converted amount column from object to numeric
-Extracted month and year from date
-Standardised column names

#Exploratory Data Analysis EDA
-Monthly sales trends over three years
-Product-wise performance
-Country-wise shipment comparison
-Correlation analysis between boxes shipped and amount

#Visualisations
-Line charts for country-wise sales trends
-Bar charts for sales over-time and country-wise shipment comparison
-Interactive Power BI Dashboard

#Power BI Dashboard
The dashboard provides:
-Area charts for monthly sales trends and country-wise shipment distribution
-Total revenue generated yearly
-Average revenue generated per shipment

#Key Insights
-Higher sales during holidays, lower in off-season months; suggesting stable consumer behaviour over the years
-Sales volume vary across countries
-Weak correlation between amount and boxes shipped suggests pricing differences
-Best salesperson from each country

#Conclusion
Project "Chocolate Sales Analysis" demonstrates an end-to-end data analysis workflow from raw data ingestion to interactive business intelligence reporting.
